# Coordenação entre sessões — Gênesis

Blackboard compartilhado entre as sessões Claude que trabalham neste projeto ao mesmo tempo.
Vive na memória do projeto (independente de branch) para todas as sessões enxergarem.

## Protocolo (toda sessão segue)
1. **Ao iniciar:** leia este arquivo antes de tocar código.
2. **Ao assumir um trabalho:** preencha/atualize sua linha em "Frentes ativas" (sessão, branch, o que está mexendo).
3. **Ao concluir um bloco / abrir PR / mergear:** atualize aqui e a data de "Última atualização".
4. **Conflito potencial:** se for tocar um arquivo listado em "Zonas de colisão", avise aqui primeiro.
