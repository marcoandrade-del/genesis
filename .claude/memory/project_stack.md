Stack definida para o projeto Gênesis:
- **Runtime**: Node.js
- **Linguagem**: TypeScript
- **ORM**: Prisma (com PostgreSQL)
- **Banco de dados**: PostgreSQL
- **Framework web**: Fastify
- **Testes**: Vitest
- **Views**: EJS + HTMX + Bootstrap 5

**Why:** Stack recomendada para um programador COBOL veterano voltando a programar — tipagem forte, schema declarativo (Prisma ≈ DATA DIVISION), código explícito.

**How to apply:** Toda geração de código deve seguir essa stack. Usar Prisma para modelagem e migrations, Fastify para rotas, TypeScript estrito.
